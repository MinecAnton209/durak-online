# Дурень Онлайн 🃏 (v4.0.0-RC.7)

Багатокористувацька онлайн-версія популярної карткової гри "Дурень", створена з нуля. Проєкт поєднує в собі чистий інтерфейс, чесну гру з серверною логікою та легкий доступ для всіх гравців.

**Live Demo:** https://durak-online-1li7.onrender.com/
**Telegram Bot:** [t.me/durakthebot](https://t.me/durakthebot)

[Read in English](README.md) | [Інструкція з встановлення](INSTALL.md)

## ✨ Можливості

*   **Багатокористувацький режим:** Грайте з 2-4 гравцями (включаючи ботів та гостей).
*   **Детальний аналіз матчів:** Післяматчевий розбір з оцінкою ходів (Найкращий хід, Помилка, Блундер) та візуалізацією гри, натхненний chess.com.
*   **Інтеграція з Telegram:** Грайте прямо в месенджері через Web App, керуйте профілем та отримуйте сповіщення.
*   **Гра без реєстрації:** Миттєво приєднуйтесь до гри, просто ввівши нікнейм.
*   **Необов'язкова реєстрація:** Створюйте акаунт для відстеження статистики, щоденних серій ("стріків"), досягнень та кастомізації.
*   **Збереження сесії:** Сайт "пам'ятає" вас навіть після оновлення сторінки.
*   **Система перепідключення:** Можливість повернутися у гру після випадкового закриття вкладки.
*   **Ігрове лобі:** Можливість створювати приватні або публічні кімнати з власними правилами (Колода 36/52, Підкидний/Перевідний).
*   **Багатомовність (i18n):** Інтерфейс доступний українською, англійською та російською мовами.
*   **Захист від шахрайства:** Вся ігрова логіка перевіряється на сервері.
*   **Адаптивний дизайн:** Підтримка мобільних пристроїв та PWA.

## 🛠️ Стек технологій

### Backend
*   **Node.js & Express.js:** Основний сервер та REST API.
*   **Socket.IO:** Двосторонній зв'язок у реальному часі.
*   **Prisma ORM:** Керування базою даних з підтримкою **SQLite** (локально) та **PostgreSQL** (продакшн).
*   **Telegraf:** Фреймворк для Telegram Bot API.
*   **Web Push:** Підтримка браузерних сповіщень.
*   **bcryptjs:** Безпечне хешування паролів.

### Frontend
*   **Vue 3 (Composition API):** Сучасний реактивний інтерфейс.
*   **Vite:** Швидка збірка та сервер розробки.
*   **Pinia:** Керування станом (state management).
*   **Tailwind CSS:** Стилізація з кастомними глассморфізм компонентами.
*   **Vue I18n:** Система локалізації.

## 🚀 Локальний запуск

Щоб запустити цей проєкт на своєму комп'ютері:

1.  **Клонуйте репозиторій:**
    ```bash
    git clone https://github.com/MinecAnton209/durak-online.git
    cd durak-online
    ```

2.  **Встановіть залежності:**
    ```bash
    pnpm install
    ```

3.  **Налаштуйте файл середовища:**
    *   Скопіюйте `.env.example` у `.env` та заповніть необхідні змінні.

4.  **Ознайомтеся з [INSTALL.md](INSTALL.md)** для отримання детальних інструкцій з налаштування бази даних.

5.  **Запустіть сервери:**
    ```bash
    # Термінал 1 (Backend)
    node server.js

    # Термінал 2 (Frontend)
    cd client && pnpm run dev
    ```
